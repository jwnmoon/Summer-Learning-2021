Stoinder(St Olaf + Tinder) Project - C++, React Native

This was a team project for my Software Design class. This is a web application with a client-server model that allows users to anonymously submit messages and display them on a feed.
There is a Facebook page called "St Olaf Flirts", which St Olaf students often use to anonymously submit "flirts" using a form, and an admin would then upload those "flirts" on a page.
Me and my team decided to create a "clone" of this page using C++ and React Native. Building this project was by far one of the hardest things I've ever did, but I learned so much out of it!
One of the most important things I learned was how to integrate the front-end and back-end to create a smooth program.
It was confusing how to make the front and back-end communicate with each other, but it worked out in the end!
I also learned how to read other people's code and use git and gitlab.
